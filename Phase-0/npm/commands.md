# NPM Commands in Phase 0

## npm Version

show what versin installed in your OS

`npm -v`
`npm --version`

## Install npm on Project

Convert a simple folder into an official npm project

`npm init`

## Fast Install npm on Project

Convert a simple folder into an official npm project With no Question

`npm init -y`

## Install Package

Installing a package

`npm install <package-name>`

## Uninstall Package

Remove installed package from project

`npm uninstall <package-name>`

## Show Installed Package List

Report the list of installed packages

`npm list`

## Help

Help how to use a command

`npm help <command>`
